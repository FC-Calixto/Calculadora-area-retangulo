# Calcula Área do Retângulo

Projeto simples em Java que solicita ao usuário as medidas da base e da altura de um retângulo, valida a entrada garantindo até duas casas decimais, e calcula a área exibindo o resultado formatado.

## Funcionalidades

- Solicita as medidas de base e altura do retângulo em metros, aceitando números decimais com até duas casas decimais.
- Normaliza entrada substituindo vírgula por ponto para facilitar a digitação.
- Valida entrada para garantir que seja número positivo e formato correto.
- Exibe a área do retângulo com duas casas decimais, seguindo o padrão métrico (m²).
- Trata entradas inválidas com mensagens de erro amigáveis.

## Como usar

1. Clone ou baixe este repositório.
2. Compile o programa: javac CalculaAreaRetangulo.java
3. Execute o programa: java CalculaAreaRetangulo
4. Informe as medidas da base e da altura quando solicitado.

## Tecnologias

- Java 8 ou superior
- Biblioteca padrão Java (java.util.Scanner)
